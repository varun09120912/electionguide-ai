## ✨ Features
- 💬 Smart AI Chat (keyword-based engine)
- 📅 Interactive Election Timeline
- 🧠 MCQ Quiz with scoring
- 👤 Voter eligibility checker
- 📱 Works on Mobile → TV (fully responsive)
- 🔒 No API keys needed

## 🛠️ Tech Stack
| Layer | Technology |
|-------|-----------|
| Frontend | React 18 + Vite |
| Styling | Tailwind CSS + Framer Motion |
| Icons | lucide-react |
| Storage | localStorage |
| Deployment | GCP Cloud Run + Firebase Hosting |
| CI/CD | GitHub Actions |

## 📦 Setup
```bash
git clone https://github.com/yourusername/electionguide-ai
cd electionguide-ai
npm install
npm run dev
```

## 🐳 Docker (GCP Cloud Run)
```bash
docker build -t electionguide-ai .
docker run -p 8080:8080 electionguide-ai
```
